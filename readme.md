# Asioiden seuranta

Päivittäinen seuranta-sovellus aamun ja illan tehtäville.

## Ominaisuudet

- Lisää vapaasti omia tehtäviä aamuun ja iltaan
- Merkitse tehtävä tehdyksi — tallentaa kellonajan
- 7 päivän historia
- Toimii suoraan selaimessa, ei vaadi kirjautumista
- Tiedot tallennetaan laitteelle
